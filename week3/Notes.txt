- MongoDB
   - NoSQL database
   - document oriented databse
   - each database server has a collection
   - we get the response in form of a object which has a key value pair 
     -- {
          acknowledged:true,
          _id:encrypted object id 
        }
   - collections instead of tables - each collection has documents instead of rows almost identical to js object
   - {
        field:value,
        field:value
      }
   - commands:
     - cls -> clears screen
     - use database_name
     - show database/ shpow dbs
     - db.createCollection("collection_name") -> takes cuurent db and creates collection -> db means current database
     - db.collection_name.insertOne({field:value}) -> inserts one document
     - db.collection_name.insertName({field:value},{field:name}) -> inserts array of documents
     - db.collection_name.findOne() -> displays first document
     - db.collection_name.findMany() -> displays all documents
     - db.collection_name.findOne({field:{condition:[value] -> called as query operator}}) -> shows first match document
     - db.collection_name.find({field:{query operator:[condition1,condition2]}}) -> shows all matched cursor -> which contains all matched document
     - db.collection_name.updateOne({query operator(where u want to change)},{$set:{modifies key value pairs}})
     - db.collection_name.updateMany({{},{}},{{},{}})
     - db.collection_name.deleteOne({condition})
     - db.collection_name.deleteMany({condition})
   - query operators:
     - $gt -> greater than 
     - $lt -> less than 
     - $eq -> equals to 
     - $in -> 
     - $and -> joins multiple conditions with query operators
     - $set -> to update
     - $unset -> to delete
     - $push -> pushes a value without checking if its already existing or not
     - $addToSet -> checks if value is already existing and does not push if already existing
     - $each -> performs operation of each and every element of the array
     - $pop -> 1 for bottom end and -1 for first element to delete 
     - $pull -> to delete using a specified field and value
     - $all -> checks irrespective of indexes
